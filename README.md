# 📄 Recruitment Screener Agent
Resumes lie. Reading them takes hours. This agent acts as your elite Technical Recruiter. It doesn't just read the resume; it actively browses the web to verify the candidate's public footprint (LinkedIn, GitHub) and contrasts the findings against your Job Description.

### ✨ Enterprise Features
* **Active Web Verification:** Cross-references resume claims with live web data.
* **Anti-Bias Guardrails:** Clinical, objective evaluations based purely on data.
* **MECE Skill Gaps:** Clear, non-overlapping identification of what the candidate lacks.
* **Tailored Interview Strategy:** Generates custom questions to probe weak spots.

### 🚀 Installation (A2A Protocol)

npx skillsadd mateusoliveiradev1/recruitment-screener
